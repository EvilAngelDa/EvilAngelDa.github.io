# Zombie Mower

一个纯静态网页小游戏原型，玩法是俯视角移动 + 自动攻击 + 僵尸围攻 + 打怪升级。

## 本地运行

直接打开 `index.html` 就能玩。

如果你想更接近真实部署环境，也可以在当前目录起一个静态服务，例如：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`

## 部署

把下面这些文件上传到你自己的服务器站点目录即可：

- `index.html`
- `styles.css`
- `game.js`
- `cover.svg`

适合放在 Nginx、Apache、宝塔静态站点，或者任何能托管静态文件的环境里。

## 当前玩法

- `WASD` / 方向键移动
- `Shift` 冲刺
- 自动朝最近敌人开火
- 击杀掉经验球，升级时三选一强化
- 随机掉落治疗球
- 被围殴到生命归零后可重开
