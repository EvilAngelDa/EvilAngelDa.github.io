# Rift Sentinel / 裂隙哨兵

一款纵向波次生存射击游戏。玩家驾驶战机守在战场底部，只需左右移动；武器会自动锁定最危险的怪物并持续攻击。

## 操作

- `A / D` 或 `← / →`：左右移动
- `P` 或 `Esc`：暂停 / 继续
- 手机和平板：按住画面底部左右按钮移动

## 游戏系统

- 自动瞄准与射击
- 普通、快速、重装和 Boss 四类怪物
- 每波按指数提升敌人数量、生成频率、生命、速度和伤害，并设置性能上限
- 快速怪会蛇形突进，越过战场中段的敌人会进入冲刺状态
- 每 5 波出现 Boss；Boss 会先显示预警线，再发射不追踪玩家的直线弹
- 经验升级与随机三选一强化
- 伤害、射速、多重弹道、穿透、暴击、移速、装甲和护盾强化
- 生命、护盾、超频三类战场补给，掉落物会被约束在战场左右边界内
- 积分、最高分、击杀、连击、波次和武器参数统计，积分采用低倍率计分
- 本地保存最高分
- 键盘、触控和指针拖动支持

## 技术

- Phaser `3.90.0`
- Arcade Physics
- HTML / CSS / JavaScript
- 游戏图形由代码实时生成，不依赖外部美术资源

直接打开 `index.html` 即可运行；使用本地静态服务器打开时兼容性更稳定。
